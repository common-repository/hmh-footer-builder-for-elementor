== Installation ==

How to install the plugin and get it working?

1. Upload the plugin files to the `/wp-content/plugins/hmh-footer-builder-for-elementor` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Sections->All Footers screen to management Footers
4. Use the Sections->Add Footer screen to create footer
